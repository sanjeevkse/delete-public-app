=== ACCESS INFORMATION ===

URL: http://localhost:3001/admin
Method: GET
Auth: Required (JWT token in Authorization header)
Format: Browser-rendered HTML (EJS template)

Steps to access:
1. Login via POST /api/auth/login to get JWT token
2. Add token to Authorization header: "Bearer <token>"
3. Navigate to http://localhost:3001/admin

=== DATABASE QUERIES ===

Dashboard loads:
- User count (SELECT COUNT(*) FROM tbl_user)
- Role count (SELECT COUNT(*) FROM tbl_meta_user_role)
- Permission count (SELECT COUNT(*) FROM tbl_meta_permission)
- Permission group count (SELECT COUNT(*) FROM tbl_meta_permission_group)
- Recent users (SELECT * FROM tbl_user ORDER BY created_at DESC LIMIT 5)

Total: 5 queries per dashboard load
